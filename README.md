# Mini RDBMS (Educational Project)

A simple relational database management system implemented in Python.

## Features
- Table creation with column types
- Primary key and unique constraints
- CRUD operations
- Basic indexing
- INNER JOIN support
- SQL-like interface
- Interactive REPL
- Flask web app demonstration

## Files
- `mini_rdbms.py` – Core database engine
- `repl.py` – Interactive REPL
- `app.py` – Flask web CRUD demo

## Run the REPL
```bash
python repl.py
